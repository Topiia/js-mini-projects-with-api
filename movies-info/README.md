A Netflix-style movie explorer using the TMDB API.

## Features
- **Trending Movies**: Shows popular movies on load.
- **Search**: Find any movie by title.
- **Modal Details**: Click any movie to see full details (Overview, Rating, Year) without leaving the page.
- **Responsive Grid**: Adapts to all screen sizes.

## API
Uses [The Movie Database (TMDB)](https://www.themoviedb.org/) API.
